# /lsp-onboard

1. `create_session`
2. `import_project(project_id, source)` — local path or URL
3. `checkout_workspace(session_id, project_id)`
4. `ensure_runtime(session_id, language)`
5. `warm_index(session_id)`
6. Ready for scout tools (`blast_radius`, `explore_symbol`, …)
